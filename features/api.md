# API

Landscape 中, 你在 UI 上的所有操作都可以通过 API 完成。

部署完成后, 访问 [https://landscape.local:6443/api/docs](https://landscape.local:6443/api/docs) 即可打开 REST API 文档。

## npm 库

此外还提供 npm 库: [@landscape-router/types](https://www.npmjs.com/package/@landscape-router/types)

## 使用该库编写的插件和辅助程序

- [landscape 浏览器分流追踪插件](https://github.com/landscape-router/landscape-browser-extension)
